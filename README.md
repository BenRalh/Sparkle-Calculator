# 🦄 Sparkle Calculator

A pastel-rainbow calculator built with **Vite + React** — complete with a draggable, animated pet unicorn, satisfying ASMR button sounds, and a handful of hidden easter eggs.

![Pastel rainbow calculator with a kawaii unicorn](https://img.shields.io/badge/vibe-✨%20whimsical-ffd1dc)

## Features

- **Full calculator** — four operations with chained evaluation, `%`, `±`, decimal, divide-by-zero handling, floating-point cleanup, and complete keyboard support.
- **3D pastel keys** — chunky, tactile buttons that press down with a satisfying squash.
- **ASMR sounds** — soft, shimmery bell-twinkles on hover and a warmer "plink" on press, all synthesized live with the Web Audio API (no audio files).
- **Sparkle the unicorn** 🦄 — a kawaii pet that:
  - flies/glides around the screen on her own and lands on the buttons
  - blinks, winks (with a sparkle), breathes, and sways her tail
  - naps on a button when idle, with floating Zzz's
  - can be **dragged**, **petted**, **fed** 🍓, and **played** with 🎈
- **Twinkling sparkles** drifting around the calculator.
- **Hidden easter eggs** — type certain numbers and see what happens. 🌿

## Getting started

```bash
npm install
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # production build
npm run preview  # preview the production build
```

## Tech

- [Vite](https://vitejs.dev/) + [React 18](https://react.dev/)
- Plain CSS animations and the Web Audio API — no extra runtime dependencies
